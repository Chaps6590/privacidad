Política de Privacidad – Bot Assistant

La presente Política de Privacidad describe cómo se recopilan, utilizan y protegen los datos personales proporcionados por los usuarios que interactúan con Bot Assistant, un asistente automatizado utilizado para brindar información, gestionar solicitudes y agendar turnos.

1. Responsable del Tratamiento de Datos

El responsable del tratamiento de los datos es el desarrollador y propietario de Bot Assistant, quien actúa de manera independiente como prestador de servicios digitales.

2. Datos que se Recopilan

Bot Assistant puede solicitar, recibir y procesar los siguientes datos personales del usuario:

  • Nombre y apellido
  • DNI u otro documento de identificación
  • Datos de contacto (teléfono, WhatsApp)
  • Consultas o información que el usuario proporcione voluntariamente
  • Datos necesarios para gestionar turnos, citas o solicitudes específicas

3. Finalidad del Tratamiento

Los datos recopilados se utilizan exclusivamente para:

  • Responder consultas realizadas por el usuario
  • Brindar información solicitada
  • Agendar turnos o citas
  • Confirmar, modificar o cancelar turnos
  • Mejorar la experiencia de uso del asistente

En ningún caso los datos se utilizan para fines comerciales no autorizados, publicidad masiva o cesión a terceros.

4. Base Legal del Tratamiento

El tratamiento de los datos se basa en el consentimiento expreso del usuario, otorgado cuando interactúa con el asistente y proporciona su información de manera voluntaria.

5. Almacenamiento y Protección de los Datos

Los datos se almacenan de forma segura y se toman las medidas adecuadas para evitar accesos no autorizados, alteración o pérdida.

El desarrollador se compromete a garantizar la confidencialidad y buen uso de la información.

6. Duración del Almacenamiento

Los datos se conservarán únicamente durante el tiempo necesario para cumplir con las finalidades descritas, o hasta que el usuario solicite su eliminación.

7. Derechos del Usuario

Los usuarios pueden ejercer los siguientes derechos sobre sus datos personales:

  • Acceder a los datos almacenados
  • Solicitar la rectificación de datos incorrectos
  • Solicitar la eliminación total de sus datos
  • Retirar el consentimiento en cualquier momento

Para ejercer estos derechos, el usuario puede comunicarse con el responsable mediante el mismo canal donde interactúa con Bot Assistant.

8. Cesión de Datos

No se comparten, venden ni transfieren datos a terceros.
Solo se utilizarán servicios tecnológicos necesarios para el funcionamiento del asistente (ej. herramientas de mensajería o automatización), siempre cumpliendo con estándares de seguridad.

9. Uso de Servicios de Terceros

Bot Assistant puede utilizar herramientas de terceros (APIs, plataformas de mensajería, sistemas de turnos) únicamente para cumplir con la finalidad del servicio.

10. Modificaciones de esta Política

El responsable puede actualizar esta Política en cualquier momento. Las modificaciones se publicarán en esta misma página.

11. Contacto

Para consultas relacionadas con esta Política de Privacidad, el usuario puede comunicarse a través del mismo canal donde utiliza Bot Assistant.
